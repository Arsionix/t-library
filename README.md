# Т-Библиотека

Консольное приложение для управления личной библиотекой.

## Возможности

### Управление книгами
- **Добавление книги** - название, автор, жанр, год издания, описание
- **Удаление книги** - с подтверждением действия
- **Поиск книг** - по ключевым словам в названии, авторе, жанре или описании

### Просмотр и сортировка
- **Список всех книг** с ID, годом и описанием
- **Сортировка** по названию, автору или году издания
- **Фильтрация** по жанру и статусу прочтения

### Избранное и статусы
- **Добавление в избранное** - отдельный список любимых книг
- **Отметка о прочтении** - отслеживание прочитанных книг
- **Отдельный просмотр** избранных книг

### Сохранение данных
- Автоматическое **сохранение в JSON-файл** после каждого изменения
- **Загрузка данных** при запуске программы
- Данные не теряются при закрытии

## Как запустить

1. Убедись, что установлен Python (версия 3.10 или выше)
2. Скачай все файлы проекта
3. Открой терминал в папке с проектом
4. Запусти программу: `python src/main.py`

## Пример использования
Загрузка библиотеки...  
Загружено 3 книг из файла

- ========================================

## Требования
- Python 3.10+
- Никаких дополнительных библиотек не требуется

## Важно!
- Все данные автоматически сохраняются в папке `data/`
- При первом запуске папка создается автоматически
- ID книг не сбрасываются при удалении (видна история действий)
