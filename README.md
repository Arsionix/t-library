# Т-Библиотека 📚

Консольное приложение для управления личной библиотекой.

## Возможности
- Добавление книг (название, автор, жанр)
- Просмотр всех книг
- Добавление в избранное 
- Отметка о прочтении 
- Удаление книг 
- Поиск книг 
- Сохранение в файл 

## Как запустить
1. Убедись что установлен Python
2. Скачай все файлы
3. Открой терминал в папке с проектом
4. Запусти: `python src/main.py`
